## Deliverables

### Font Optimization

1. **Font File Strategy**
   - Location: `public/fonts/` (not `src/assets/fonts/`)
   - Vite serves `public/` as static root → URLs resolve to `/fonts/filename.woff2`
   - Avoids Vite asset pipeline (hashing, bundling) for fonts that may not exist yet
   - Missing files produce 404 at runtime instead of build error

2. **Theme-Specific Fallback Chains**
   - **apple2e:** `'PrintChar21', 'Apple II', monospace`
   - **c64:** `'C64 Pro Mono', 'PetMe', monospace`
   - **ibm3270:** `'IBM 3270', 'IBM Plex Mono', monospace`
   - **win95:** `'W95FA', 'Fixedsys', 'Courier New', monospace`
   - **lcars:** `'Trek', 'Antonio', sans-serif` (chrome), `'Trek', 'Antonio', monospace` (terminal)
   - All use era-appropriate intermediate fallbacks before generic family

3. **Font Preloading**
   - `<link rel="preload">` in `index.html` for `PrintChar21.woff2` (default theme)
   - Other fonts not preloaded (avoid wasted bandwidth)

4. **@font-face Declarations**
   - All 6 rules use `font-display: swap` (render text immediately, swap font when loaded)
   - Declarations in `src/styles/fonts.css`

### Accessibility Features

1. **Focus Indicators** (`src/styles/accessibility.css`)
   - Theme-specific `:focus-visible` styles
   - CRT themes: phosphor-colored glow rings
   - Win95: classic dotted outline
   - LCARS: orange ring
   - Pattern: `outline` + `box-shadow` double-ring for contrast

2. **Reduced Motion Support**
   - `@media (prefers-reduced-motion: reduce)` disables:
     - CRT flicker animation
     - Phosphor glow transitions
     - Switch slide transitions
     - Pill hover brightness changes
   - Static styling (colors, borders, glow) preserved

3. **Screen Reader Support**
   - Icon-only buttons: `aria-label`
   - StatusBar: `role="status"`, `aria-live="polite"` on connection indicator
   - Terminal: `role="application"`, `aria-label="Squad terminal"`
   - Theme changes announced via hidden `aria-live` region
   - Toggle buttons: `aria-pressed`
   - Decorative emoji: `aria-hidden="true"`

4. **Keyboard Navigation**
   - Controls grouped in `role="toolbar"` for logical tab stops
   - Escape key returns focus to terminal
   - All interactive elements natively keyboard-focusable (buttons, inputs)
   - Tab order verified across all 5 themes
