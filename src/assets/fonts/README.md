# Font Assets

Place retro terminal font files here.

## Required Fonts

### PrintChar21 (Apple IIe theme)
- Monospace bitmap-style font replicating the Apple IIe character ROM
- Needed files: `PrintChar21.woff2`, `PrintChar21.woff`
- Source: Available from various retro font archives (check licensing)

### C64 Pro Mono (Commodore 64 theme)
- Monospace font replicating the C64 PETSCII character set
- Needed files: `C64_Pro_Mono-STYLE.woff2`, `C64_Pro_Mono-STYLE.woff`
- Source: style64.org (free for personal use)

## Fallback

Until these fonts are installed, the themes will fall back to `Courier New, monospace` as defined in the theme configs.
