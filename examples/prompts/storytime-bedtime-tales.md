# Storytime: Bedtime Tales — Prompts

## Tell me a bedtime story
Create a cozy bedtime story for a young child. Ask me a couple of questions first so you can make it just right.

## Create another story like this one
Write a new bedtime story with a similar feel and style, but a completely different adventure.

## Create another story with the same characters
Write a new bedtime story using the same characters from the last story, but give them a brand new adventure.

## Make it a little shorter
Retell the last story in a shorter version — about 3–5 minutes to read aloud.

## Add a lullaby at the end
Write a short, gentle lullaby or rhyme that goes with the story I can sing or read as a final goodnight.
